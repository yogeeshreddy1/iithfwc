\paragraph{\textit{Problem Statement} - Suppose the cubic  $P (x) = x^3-px+q$ has three distinct real roots where p$>$0 and q$>$0.Then which of the following holds?  }


\section*{\large Solution}
\begin{align}
P (x) = x^3-px+q
\end{align}


$P'(x) = 3x^2-p$

 
f(x)  consists of minima and maxima,\\
Using gradient ascent method we can find its minima ,
    \begin{align}
        x_{n+1} &= x_n - \alpha \nabla f(x_n) \\
        \implies x_{n+1} &= x_n - \alpha \brak{3x_n^2-1}
    \end{align}
Using gradient descent method we can find its maxima ,
    \begin{align}
        x_{n+1} &= x_n + \alpha \nabla f(x_n) \\
        \implies x_{n+1} &= x_n + \alpha \brak{3x_n^2-1}
    \end{align}    
    
Taking $p=1,q=1,x_0=0.1,\alpha=0.001$ and precision = 0.00000001, values obtained using python are:    
    \begin{align}
        \boxed{\text{Minima} = 0.6154}\\
        \boxed{\text{Minima Point} = 0.5777}
    \end{align}
     
    \begin{align}
        \boxed{\text{Maxima} = 1.3846}\\
        \boxed{\text{Maxima Point} = -0.5777}
    \end{align}
